Git Tutorial

This tutorial will guide you through the basic commands to use Git for version control.

1. Install Git

Download and install Git from the official Git website.

2. Configure Git

After installation, configure your Git username and email:

git config --global user.name "Your Name"
git config --global user.email "youremail@example.com"

3. Initialize a Repository

Navigate to your project directory and run:

git init

This creates an empty Git repository.

4. Check the Status

To check the current status of your repository, use:

git status

5. Add Files to Staging

To add files to the staging area, use:

git add filename

Or, to add all files:

git add .

6. Commit Changes

To save your changes, commit them with a message:

git commit -m "Your commit message"

7. Connect to a Remote Repository

To connect to a remote repository (e.g., GitHub):

git remote add origin https://github.com/username/repository.git

8. Push Changes

To push your changes to the remote repository:

git push -u origin main

9. Pull Changes

To pull updates from the remote repository:

git pull

10. Clone a Repository

To clone an existing repository:

git clone https://github.com/username/repository.git
